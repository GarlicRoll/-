# - Бизнес процесс управления ресторанной сетью на основе фильма "Основатель"
1. Фильм демонстрирует три ключевые фазы эволюции бизнес-процесса (соответствует фазам жизненного цикла UP):

| Фаза | Состояние бизнес-процесса | Изменения | Сложность |
| ---- | --------------------------| --------- | --------- |
| Начало (братья Макдональдс) | Локальный ресторан с ручной оптимизацией "Системы Спиди": потоковая кухня, ограниченное меню, самообслуживание | Фокус на операционной эффективности одного объекта | Низкая (1 ресторан, 1 процесс) |
| Уточнение (встреча с Рэем Кроком) | Стандартизация процессов: единые рецептуры, оборудование, планировка | Переход от уникального решения к повторяемому шаблону | Средняя (требуется документирование и контроль) |
| Построение (франчайзинговая сеть) | Децентрализованная сеть с централизованным контролем: франшизы, обучение, аудит качества, управление недвижимостью | Внедрение многоуровневой архитектуры управления | Высокая (масштабирование + контроль качества) |

Ключевое изменение в бизнес-процессе: переход от производственной модели (продажа гамбургеров) к платформенной (продажа франшиз + контроль через недвижимость). Это соответствует подходу в UP - фундаментальная перестройка системы.

2. Роли (актеры) в бизнес-процессе
На основе фильма выделяем 5 ключевых актеров (согласно разделу 4.3.2):

| Роль | Описание | Вклад в бизнес-процесс |
| ---- | -------- | ---------------------- |
| Франчайзи |	Владелец отдельного ресторана, покупающий право на использование бренда |	Финансирование локальной операции, соблюдение стандартов, генерация роялти |
| Инспектор качества | Представитель головной компании, контролирующий соответствие стандартам | Обеспечение консистентности бренда, предотвращение девиаций в процессах |
| Оператор ресторана | Менеджер ежедневной работы (график, инвентарь, персонал) | Реализация операционных прецедентов на уровне точки продаж |
| Центр снабжения |	Отдел закупок и логистики головной компании | Централизация закупок для снижения себестоимости (как в фильме с сиропом) |
| Разработчик стандартов | Эксперт по оптимизации процессов (роль братьев Макдональдс на старте) | Создание и улучшение «Системы Спиди» - ядра бизнес-процесса |

3. Функциональные требования к системе управления

Согласно главе 4.5, требования формулируем как прецеденты с акцентом на что, а не как:

    1. Регистрировать франшизу - оформление договора и передача стандартов
    2. Проводить аудит качества - проверка соответствия операционным стандартам
    3. Управлять обучением персонала - сертификация сотрудников по "Системе Спиди"
    4. Контролировать закупки - централизованный заказ ингредиентов и оборудования
    5. Рассчитывать роялти - автоматизированный сбор платежей от франчайзи
    6. Управлять недвижимостью - аренда/покупка участков под рестораны (ключевая инновация Крока)
    7. Стандартизировать меню - введение/изменение позиций с контролем локальной адаптации
    8. Анализировать производительность - отслеживание KPI (время обслуживания, отходы)
    9. Обрабатывать жалобы клиентов - централизованная система обратной связи
    10. Планировать расширение сети - географический анализ и выбор локаций
    11. Управлять поставщиками - аккредитация и контроль качества сырья
    12. Разрабатывать оборудование - проектирование специализированных станков (как в фильме с картофелем)
    13. Контролировать гигиену - аудит санитарных норм на кухне
    14. Управлять маркетингом - локальные и глобальные рекламные кампании
    15. Вести финансовую отчетность - консолидация данных от всех точек сети
